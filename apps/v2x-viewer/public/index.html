Documentation

          

            

              Data is stored in DynamoDB table v2x_detections with keys:
              object_id (PK) and ts_event (SK =
              timestamp_utc#event_id). A GSI exists on
              geohash + ts_event.

              Notes
            

            
- This UI is intentionally small and read-only.
